🔍 Diagnóstico de Ollama

📋 Comandos para verificar en tu terminal:

# Verificar si Ollama está instalado
ollama --version

# Listar modelos descargados
ollama list

# Iniciar servidor Ollama (si no está corriendo)
ollama serve

# En otra terminal, descargar un modelo (si no tienes ninguno)
ollama pull llama3.2
